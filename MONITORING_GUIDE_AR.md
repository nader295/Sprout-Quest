# 📈 دليل مراقبة استهلاك Function Invocation

## 🔍 كيفية متابعة الاستهلاك:

### الطريقة 1: لوحة Vercel الرئيسية

1. اذهب إلى: **https://vercel.com/dashboard**
2. اختر مشروعك **nadersgs-projects**
3. اضغط على **Analytics** أو **Monitoring**
4. ابحث عن **Function Invocations**

### الطريقة 2: التفاصيل الدقيقة

1. **Settings** → **Monitoring**
2. **Function Invocations** → اضغط على الـ chart
3. سترى:
   - إجمالي الاستدعاءات اليومية
   - أكثر endpoints استهلاكاً
   - الـ functions الباهظة

---

## 📊 ما الذي تبحث عنه:

### الأرقام الصحية:

## 🎯 أهداف قياسية:

بعد التطبيقات:
- **الهدف 1:** تقليل من 1,000,000 إلى < 700,000 (30 يوم من الآن)
- **الهدف 2:** تقليل من 700,000 إلى < 500,000 (60 يوم)
- **الهدف 3:** تقليل من 500,000 إلى < 300,000 (90 يوم)

## 🔧 الأوامر للمراقبة (إذا كنت في Terminal):

### استخدام Vercel CLI:

```bash
# تثبيت Vercel CLI
npm i -g vercel

# تسجيل الدخول
vercel login

# عرض الإحصائيات
vercel analytics --project=nadersgs-projects

# عرض الـ logs
vercel logs --project=nadersgs-projects
```
